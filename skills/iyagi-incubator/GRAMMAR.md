# Known Korean Grammar

List grammar formulas the learner already knows. Keep one formula per row. Wrap formulas and examples in backticks so they are easy to scan.

| Formula | Meaning / Use |
|---|---|
| `N-이에요/예요` | "To be"; identifies or equates a noun with something. |
| `N-이/가 아니에요` | "To not be"; negates identity. |
| `N-은/는` | Topic particle; marks the topic of a sentence. |
| `N-이/가` | Subject particle; marks the grammatical subject. |
| `있어요 / 없어요` | Expresses existence, possession, or absence. |
| `V-(아/어) 주세요` | Polite request asking someone to do something. |
| `V-고 싶어요` | Expresses wanting to do an action. |
| `V-고 싶어요?` | Asks what someone wants to do. |
| `V-아/어/여요` | Basic present tense polite verb ending. |
| `V-았/었/였어요` | Basic past tense polite verb ending. |
| `N-에` | Marks a destination, time, or static location. |
| `N-에서` | Marks the place where an action occurs. |
| `V-안` | Short-form negation placed before a verb. |
| `V-지 않다` | Long-form negation meaning "do not." |
| `V-하다` | Basic "to do" verb used to form many action verbs. |
| `N-에서/부터` | Indicates the starting point in place or time. |
| `N-까지` | Indicates an ending point or limit. |
| `V-고` | Connects actions or clauses; "and," "then." |
| `N-(이)랑` | Connects nouns meaning "and" or "with." |
| `N-하고` | Connects nouns meaning "and" or "with." |
| `N-을/를` | Object marking particle. |
| `N-한테` | Indicates the recipient of an action ("to someone"). |
| `N-한테서` | Indicates the source of an action ("from someone"). |
| `V-고 있어요` | Present progressive; "be doing." |
| `N-개, 명, etc.` | Counters used when counting objects, people, and other items. |
| `A/V-지만` | Expresses contrast; "but," "although." |
| `A/V-는데` | Gives background, contrast, or sets up the next statement. |
| `N-인데` | Gives background or contrast after a noun. |
| `V-(으)ㄹ 거예요` | Indicates future plans, intentions, or predictions. |
| `N-도` | Means "also," "too," or "even." |
| `N-만` | Means "only." |
| `V-(으)ㄹ 수 있다` | Expresses ability or possibility; "can." |
| `V-(으)ㄹ 수 없다` | Expresses inability or impossibility; "cannot." |
| `A-잘하다` | Means to be good at doing something. |
| `A-못하다` | Means to be bad at or unable to do something. |
| `V-는 것` | Turns a verb into a noun ("the act of doing..."). |
| `V-아/어야 되다` | Expresses obligation; "have to," "must." |
| `V-아/어야 하다` | Expresses obligation; "have to," "must." |
| `A-보다 더` | Comparative expression meaning "more than." |
| `A-좋다` | Means "to be good." |
| `V-좋아하다` | Means "to like." |
| `V-(으)면` | Expresses a condition; "if," "when." |
| `V-(으)ㄹ까요?` | Asks for suggestions, opinions, or expresses wondering. |
| `N-쯤` | Indicates approximation; "around," "about." |
| `N-정도` | Indicates degree or approximation. |
| `N-약` | Means "approximately" or "about." |
| `V-(으)ㄹ래요` | Expresses the speaker's intention or asks someone's preference. |
| `V-(으)ㄹ래요?` | Asks about someone else's will or intention; often "Do you want to...?" |
| `V-(으)ㄹ게요` | Expresses a decision or promise made at the moment of speaking. |
| `V-아/어/여서` | Connects clauses meaning "because" or "and then." |
| `A-같다` | Means "seem" or "look like." |
| `V-(으)ㄴ 것 같다` | Indicates that something seems to be true (past/descriptive). |
| `V-는 것 같다` | Indicates that something seems to be happening (present). |
| `V-(으)ㄹ 것 같다` | Indicates that something seems likely to happen (future). |
| `V-기 전에` | Means "before doing." |
| `V-ㅂ/습니다` | Formal polite speech style. |
| `A-(으)ㄴ + N` | Turns an adjective into a noun modifier. |
| `V-는 + N` | Turns a present-tense verb into a noun modifier. |
| `V-아/어요 (청유형)` | Inclusive suggestion; "Let's..." |
| `V-기 위해(서)` | Means "in order to." |
| `N-밖에 + 부정` | Means "nothing but" or "only," used with a negative verb. |
| `V-다음에` | Means "after doing" or "next." |
| `V-후에` | Means "after doing." |
| `V-(으)려고` | Expresses intention or purpose; "in order to." |
| `A/V-그리고` | Connects ideas meaning "and." |
| `A/V-그래서` | Connects ideas meaning "so" or "therefore." |
| `A/V-그렇지만` | Means "however" or "but." |
| `A/V-그런데` | Introduces contrast or unexpected information. |
| `A-너무` | Means "too" or "very," depending on context. |
| `N-앞에` | Means "in front of." |
| `N-뒤에` | Means "behind." |
| `N-옆에` | Means "next to." |
| `N-위에` | Means "on top of." |
| `N-밑에` | Means "under" or "beneath." |
| `V-(으)세요` | Polite imperative or honorific request. |
| `N/V-(으)로` | Indicates direction, method, means, or choice. |
| `N-다` | Means "all." |
| `N-더` | Means "more." |
| `V-지 마세요` | Polite negative command; "Don't do..." |
